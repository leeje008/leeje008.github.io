주요 기능

  • 최적 동선 산출: Google OR-Tools 기반 TSP 경로 최적화
  • 주변 POI 탐색: 경로 주변 카페/맛집 등 자동 탐색 및 랭킹
  • LLM Agent 패키지: LangGraph 기반 스토리텔링 + 추천 이유 생성
  • 지도/타임테이블: Folium 지도 + 타임라인 시각화
  • 피드백 수집: 사용자 피드백으로 추천 품질 개선
