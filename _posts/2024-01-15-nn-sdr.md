## 프로젝트 개요

고차원 설명변수 $X \in \mathbb{R}^p$에서 반응변수 $Y$에 대한 정보를 보존하면서 차원을 축소하는 Sufficient Dimension Reduction(SDR) 문제를 뉴럴 네트워크로 해결하는 연구 구현체이다. 기존 역회귀 기반 추정기(SIR, SAVE, DR)의 선형성 가정을 극복하고, group lasso 페널티를 통해 구조적 차원 $d$를 자동 선택한다.

> GitHub: [leeje008/NN_SDR](https://github.com/leeje008/NN_SDR)

## 연구 배경

### Sufficient Dimension Reduction이란

SDR의 목표는 다음 조건을 만족하는 최소 차원의 투영 행렬 $B \in \mathbb{R}^{p \times d}$를 찾는 것이다:

$$
Y \perp X \mid B^T X
$$

즉, $B^T X$만으로 $Y$에 대한 모든 회귀 정보를 보존할 수 있다. 이때 $B$의 열공간을 **Central Mean Subspace (CMS)**라 하고, $d$를 **구조적 차원(structural dimension)**이라 한다.

### 기존 방법의 한계

| 방법 | 원리 | 한계 |
|------|------|------|
| **SIR** (Sliced Inverse Regression) | $E[X \mid Y]$의 역회귀 | 선형 조건(linearity condition) 필요 |
| **SAVE** (Sliced Average Variance Estimation) | $\text{Var}(X \mid Y)$의 변동 | 상수 분산 조건(constant covariance) 필요 |
| **DR** (Directional Regression) | SIR + SAVE 결합 | 여전히 모멘트 기반 제약 |
| **MAVE** (Minimum Average Variance Estimation) | 비모수적 접근 | 계산 비용 높음, 차원 선택 별도 |

이 방법들은 모두 **역회귀(inverse regression)** 기반으로, $E[X \mid Y]$나 $\text{Var}(X \mid Y)$를 슬라이스별로 추정한다. 비선형 link function이 강한 경우 성능이 저하되며, 구조적 차원 $d$를 별도로 결정해야 한다.

### NN-SDR의 접근

뉴럴 네트워크의 **첫 번째 레이어 가중치 행렬** $W \in \mathbb{R}^{d \times p}$가 곧 SDR 방향을 학습하도록 설계한다. 직교 제약과 group lasso를 결합하여 역회귀 없이 CMS를 직접 추정하면서 차원도 자동 선택한다.

## 수학적 정형화

### 목적 함수

$$
\mathcal{L}(\theta) = \underbrace{\frac{1}{n} \sum_{i=1}^{n} (y_i - f_{NN}(x_i))^2}_{\text{MSE}} + \underbrace{\lambda_0 \|W^T \Sigma W - I_d\|_F^2}_{\text{직교 제약}} + \underbrace{\lambda_1 \sum_{j=1}^{h_2} \|w_j^{(2)}\|_2}_{\text{Group Lasso}}
$$

각 항의 역할:

**1) MSE (예측 오차)**: 네트워크가 $Y$를 잘 예측하도록 학습. 이것만으로는 $W$가 SDR 방향이 되는 것을 보장하지 못한다.

**2) 직교 제약** ($\lambda_0$ 항): $W^T \Sigma W = I_d$를 강제한다. 여기서 $\Sigma = \frac{1}{n} X^T X$는 표본 공분산 행렬이다. 이 제약은 단순히 $W^T W = I$와 다르다. **데이터의 공분산 구조를 반영한 직교성**을 요구하여, SDR 관점에서 통계적으로 의미 있는 정규화를 수행한다.

**3) Group Lasso** ($\lambda_1$ 항): 두 번째 은닉층의 가중치를 열 단위로 정규화한다. $\|w_j^{(2)}\|_2$가 0이 되면 해당 은닉 유닛이 비활성화되므로, 활성 유닛 수가 곧 추정된 구조적 차원 $\hat{d}$가 된다.

### 네트워크 구조

```
X ∈ ℝᵖ → [W ∈ ℝᵈˣᵖ] → tanh → [H₂] → ReLU → [H₂] → tanh → [1] → ŷ
         ↑ SDR 방향          ↑ Group Lasso 적용
```

- **Layer 1**: $W \in \mathbb{R}^{d \times p}$ — SDR 방향 행렬 (직교 제약 적용)
- **Layer 2~3**: 비선형 link function $g(\cdot)$ 학습 ($h_2 = 20$ 은닉 유닛)
- **Layer 4**: 스칼라 출력

### 2단계 최적화 (매 에폭)

**Step 1 — Adam Gradient Descent**: MSE + 직교 제약의 smooth 부분을 최적화

$$
\theta^{(t+1)} = \theta^{(t)} - \eta \nabla_\theta \left[\text{MSE} + \lambda_0 \|W^T \Sigma W - I_d\|_F^2 \right]
$$

**Step 2 — Proximal Operator**: Group Lasso의 non-smooth 부분에 대해 soft-thresholding 적용

$$
w_j^{(2)} \leftarrow \left(1 - \frac{\eta \lambda_1}{\|w_j^{(2)}\|_2}\right)_+ \cdot w_j^{(2)}
$$

이 proximal step이 열 단위로 가중치를 0으로 수축시켜 **구조적 차원을 자동 선택**한다.

**수렴 제어**: 개선이 없으면 학습률을 0.9배로 감쇠시키고, $\eta < 10^{-20}$이면 조기 종료한다.

## 시뮬레이션 설계

설명변수 $X \sim \mathcal{N}(0, \Sigma)$, $\Sigma_{ij} = 0.5^{|i-j|}$ (AR(1) 공분산 구조).

### 단일 인덱스 모델 ($d = 1$)

| 모델 | 반응 함수 | 특징 |
|------|----------|------|
| M1 | $y = \exp(X^T\beta_1) + 0.5\varepsilon$ | 지수 link, $\beta_1 = (1, 0.5, 1, 0, \ldots)$ |
| M2 | $y = 0.5(X^T\beta_1)^2 + 0.5\varepsilon$ | 이차 link, $\beta_1 = (1, 1, 1, 1, 0, \ldots)$ |
| M3 | $y = \arcsin\left(\frac{1}{1+\|0.5+X^T\beta_1\|}\right) + 0.2\varepsilon$ | 비단조 link |
| M4 | $y = \sin(X^T\beta_1) + 0.5\varepsilon$ | 주기 link, $\beta_1 = (1, 0, 1, 0, \ldots)$ |

### 다중 인덱스 모델 ($d = 2$)

| 모델 | 반응 함수 | 특징 |
|------|----------|------|
| M5 | $y = \frac{d_1}{0.5 + (d_2 + 1.5)^2} + 0.5\varepsilon$ | 비선형 상호작용 |
| M6 | $y = \cos(2d_1) - \cos(d_2) + 0.2\varepsilon$ | 삼각함수 조합 |
| M7 | $y = \sin(1.4d_1) + (d_2 + 1)^2 + 0.4\varepsilon$ | 가법 구조 |
| M8 | $y = d_1^2 + d_2^2 + 0.5\varepsilon$ | 이차 가법 |

$d_j = X^T\beta_j$는 진짜 SDR 방향으로의 투영값이다.

### 평가 지표

- **Multiple Correlation**: 추정된 투영과 진짜 투영 간의 정준 상관 분석
- **Frobenius Norm**: $\|P_{\hat{B}} - P_B\|_F$ (투영 행렬 간 거리)
- **Correlation Loss**: $1 - \frac{1}{d} \text{tr}[(M_1^T\Sigma M_1)^{-1}(M_1^T\Sigma M_2)(M_2^T\Sigma M_2)^{-1}(M_2^T\Sigma M_1)]$

### 벤치마크

rpy2를 통해 R의 고전적 SDR 방법 (SIR, SAVE, DR, MAVE, rMAVE, OPG) 6개와 비교 실험을 수행한다. 각 모델에 대해 $p = 20$, $n = 500$ 조건에서 Ray Tune 기반 5-fold 교차검증으로 최적 하이퍼파라미터를 탐색했다.
